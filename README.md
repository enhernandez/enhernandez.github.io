# enriquehernandez.eu

Personal academic website of [Enrique Hernández](https://enriquehernandez.eu),
Associate Professor of Political Science at the Autonomous University of Barcelona.

Static site, hosted on GitHub Pages, served at `enriquehernandez.eu`.

## Files
- `index.html` — about / home
- `publications.html` — full list of publications with filters and abstracts
- `projects.html` — current and past funded research projects
- `outreach.html` — public-facing writing and media coverage
- `teaching.html` — courses and supervision
- `cv.pdf` — CV
- `picture.jpeg` — profile photo
- `CNAME` — GitHub Pages custom domain config
